What's Inside

AI Logic: LightGBM v1.2

Trained on 148,812 events. 63.47% accuracy. Auto-retrains with rollback safety.

UI: GTK4 Desktop

Full window manager with GoalPanel, TopBar clock, splash screen, and snap geometry.

Agents: 5 AI Agents

GoalEngine, ContextAgent, ResourceAgent, SyncDaemon, Hub. Always-on, ports 7735--7750.

Mind: MIND v2

ChromaDB vector memory, proactive monitor, voice input, NL shell commands.

Sync: Cross-Device Sync

zeroconf peer discovery. Syncs goals and context across all your devices.

ISO: Bootable ISO

Full Debian base. GRUB + UEFI. Boots in VirtualBox or real hardware.
